DRC GALLERY — how to add or change "In Action" photos.

Drop image files (.jpg / .png / .webp) into THIS folder. They appear in the
website's "In Action" gallery automatically (in filename order) the next time
the site is built/deployed. To remove a photo, delete its file. No code changes.

Tip: keep files reasonably small (long edge ~1600px). Name them so they sort the
way you want, e.g. 01-bench.jpg, 02-run.jpg, 03-group.jpg.
